About This Project

This web app is designed to help users easily track their LeetCode performance, ranking, and progress in a simple and user-friendly way. It provides a quick overview of your coding journey, making it easier to stay consistent and improve problem-solving skills. The goal of this project is to simplify access to your stats and present them in a clean dashboard-like experience.

How to Use ?

  1. ☛   Click on Setup option and select “Request Temporary Access” to enable API access.
  2. ☛   Enter your LeetCode username, and the app will display your ranking and progress.
